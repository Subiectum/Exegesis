# 思虑
* 訾 (zī)：考虑；不訾前后
# 阐释
* 训：解释词义；鲁申公为诗训故
* 诂 (gǔ)：用今语解释古语
# 测算
* 拟：揣测
* 億：预料猜测
* 迎：预测、推算；迎日推筮
* 𡁕(zá)喋：深算
* 揆：度量
* 揣：度量
* 摓 (féng)：两手分而数之，摓策定数
* 选 (suàn)：计算
* 詶 (chóu)：计算
* 訾 (zī)：衡量、计量；訾粟而赋
# 总估
* 揳 (xié)：估量
* 计：估计总计；计今人之数不若死者多
# 研究
* 揅 (yán)
* 翫 (wán)：研习
# 规划
* 摹：规摹弘远
* 拟：计划
# 记忆
* 誌：记住；一经目辄誌于心
* 服膺：道理牢记在心，拳拳服膺
# 遗忘
* 詄 (dié)：遗忘；詄荡荡（忘记得荡然无存）
# 怀揣
* 赍 (jī)：赍志而没
# 梦望
* 来：梦
* 迟 (zhì)：希望；朕思迟直士
* 喁喁 (yóng)：众人向慕
* 喁望：渴望
# 理知
* 达：通晓
* 喻：明白知道；君子喻于义
* 详：详细知道；又非君所详
* 话：出善言以晓喻民之不循教者；乃话民之弗率
